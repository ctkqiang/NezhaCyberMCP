## 网络安全

**熟练度：85%**

具备系统性的网络安全攻防实践经验，涵盖漏洞挖掘、渗透测试、恶意软件分析与数字取证。熟悉常见 Web 漏洞（OWASP Top 10）的原理与利用方式，能够独立开发安全工具并进行漏洞赏金研究。深入理解 TCP/IP 协议栈与网络通信机制，具备后门原理研究与安全防护方案设计能力。

### 核心能力

- **渗透测试**：Web 应用渗透、网络扫描、漏洞利用与报告撰写
- **漏洞研究**：OWASP Top 10 漏洞原理分析，CVE 公告跟踪与影响评估
- **安全工具开发**：Python / Go 编写自动化安全扫描工具与漏洞赏金辅助工具
- **恶意软件分析**：后门通信机制研究，TCP 隐蔽通道原理实现与分析
- **数字取证**：安卓设备数据取证，符合司法机构取证规范的数据提取与完整性验证
- **安全资讯聚合**：自动化采集 GitHub Advisory、MyCERT、CVE 等多源安全公告
- **密码学应用**：密码生成、哈希验证、数据加密基础实践

### 专项领域

- **漏洞赏金**：独立开发图形化渗透测试工具集，集成多种常用安全测试模块
- **安全平台开发**：哪吒 AI 全方位网络安全平台，提供 URL / 文件恶意软件扫描与数据泄露检测
- **安全公告自动化**：NezhaCyberMCP，对接 GitHub Advisory API 与 MyCERT 门户，实现多源安全情报自动采集与持久化
- **后门原理研究**：基于 Erlang 构建高并发 TCP 后门模拟框架，用于安全教育与防御研究
- **移动端取证**：灵取证，专为执法部门设计的安卓设备数据取证工具
